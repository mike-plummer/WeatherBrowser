Weather Browser

Example of client-side geospatial visualization using TurfJS

  • Begin

Description

The map is created using Leaflet, a super mapping library for JavaScript. After the data is plotted all the number crunching is done using TurfJS, an extension to Leaflet developed by MapBox. All of this work is done client-side which means there's zero horsepower being used on your server outside of serving up static content resources. The data is served up as GeoJSON data which is easily digested by Turf. The map itself is served up by OpenStreetMap by way of MapBox.

Next

Map

Use the slider at the top to control the display of data. For performance reasons only a single year at a time can be displayed. The Layers control in the bottom left allows you to enable/disable:

  • Tornado layer - map of a given year's tornado events. Each has an on-click tooltip for more info.
  • Yearly Heat Map - heat map of the given year's tornado events. This will live-update as the is changed.
  • Overall Heat Map - heat map of all tornado events.

For aggregation information click a state. The state will highlight after a little number-crunching at which point the highlight can be clicked again for a tooltip displaying aggregate information on tornado data in that state over time.
